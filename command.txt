************************************************************************************************************************
*                                                 GIT COMMANDS
************************************************************************************************************************


------------------------------------------------------------------------------------------------------------------------
git status              :   Show how many file are changed in working repository.
git status -s           :   When we add -s switch then it show file status in left side
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git add fileName        :   Add specific file in git
git add --all           :   Add all files in git
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git commit              :   Commit Changes in git, when we use this command then one vim prompt open for write comments
git commit -m "comments":   Commit Changes in git, the -m switch and "" area provide at a time commit comments no need
                            vim editor to edit comments
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git log                 :   Show all commit log. Here we get 1.commit hash, 2.That commit Author, 3.Date, 4.comments
git log -2              :   Show last 2 commit log. When we have need see last 2 or 3 log the we using this -2 or -3
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git show commitHash     :   Show that commit changes
Example                 :   git show bf27ddb98278c96c6d4e2bb59afad9e2f7974829
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git diff                :   Show how many changes are occur in repository
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git push                :   Push single Changes in repository
git push --all          :   Push all Changes in repository
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git pull                :   Pull Others developer changes from repository and update working copy
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git clone               :   Download remote or repository for working copy
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git branch              :   Show how many branches are available in that repository
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git stash               :   When we not want to commit or push our half done code then we can save it by stash. After
                            Solve our problem then we also can back our previous half done code
git stash list          :   Show list of stash are saved
git stash pop           :   Add again our hidden code in working copy
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git rm fileName         :   Remove file from git repository
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git mv src dst          :   Rename file source to destination name
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git checkout fileName      :   Revert uncommitted changes
git checkout commitHash    :   Then the repository will go that commit
git checkout branchName    :   Switch branch
git checkout -b branchName :   Create new branch and switch branch
------------------------------------------------------------------------------------------------------------------------

------------------------------------------------------------------------------------------------------------------------
git branch                 :   Shows branches and indicate * by selected branch
git branch branchName      :   For create a new branch
git branch -D branchName   :   Delete Branch
git branch -m old new      :   Rename Branch old name to new name
------------------------------------------------------------------------------------------------------------------------
